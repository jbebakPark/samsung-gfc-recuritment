**파일**: `admin/applications.js`

3. 박민수 - 직접 지원 / 검토중
4. 최지혜 - 채용설명회 / 대기중
5. 정우성 - 추천인 경로 / 거부됨

**결과**: ✅ 지원자 관리 페이지가 정상 작동합니다.

## 📝 사용 가이드

### 관리자 페이지 접속
1. 브라우저에서 `admin/index.html` 열기
2. 네비게이션에서 원하는 메뉴 선택

### 지원자 관리
1. "지원자 관리" 메뉴 클릭
2. 샘플 데이터 5명 확인
3. 필터/검색으로 원하는 지원자 찾기
4. "상세" 버튼으로 지원서 전체 내용 확인
5. 승인/거부 버튼 클릭 (현재는 Alert만 표시)

### 인터뷰 관리
1. "인터뷰 관리" 메뉴 클릭
2. 카테고리 탭으로 필터링 (전체/언론사/포털/유튜브/블로그)
3. 검색창으로 키워드 검색
4. 정렬 옵션 선택 (최신순/오래된순/제목순)
5. "보기" 버튼으로 원본 링크 열기
