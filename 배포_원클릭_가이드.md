## 🎯 이 스크립트가 하는 일

**60분 걸리던 수동 배포를 3-5분으로 단축!** ⚡

1. ✅ 필요한 프로그램 자동 확인 (Git, Node.js, npm)
2. ✅ Firebase CLI 자동 설치
3. ✅ 프로젝트 자동 다운로드
4. ✅ Firebase 로그인 (브라우저 자동 열림)
5. ✅ 배포 전 파일 자동 검증
6. ✅ Firebase에 자동 배포
7. ✅ 배포 완료 확인
8. ✅ 결과 URL 안내

## 📥 다운로드

### Windows 사용자
**파일 이름**: `6-one-click-deploy.bat`

**다운로드 링크**:
```
https://raw.githubusercontent.com/jbebakPark/samsung-gfc-recuritment/main/scripts/6-one-click-deploy.bat
```

**다운로드 방법**:
1. 위 링크를 브라우저에 복사
2. 우클릭 → "다른 이름으로 저장"
3. 바탕화면에 저장

### Mac / Linux 사용자
**파일 이름**: `6-one-click-deploy.sh`

**다운로드 링크**:
```
https://raw.githubusercontent.com/jbebakPark/samsung-gfc-recuritment/main/scripts/6-one-click-deploy.sh
```

**다운로드 방법** (터미널):
```bash
# 홈 디렉토리에서
cd ~
curl -O https://raw.githubusercontent.com/jbebakPark/samsung-gfc-recuritment/main/scripts/6-one-click-deploy.sh
chmod +x 6-one-click-deploy.sh
```

## ▶️ 사용 방법

### 🪟 Windows

1. **다운로드한 파일 더블클릭**
   - `6-one-click-deploy.bat` 파일 더블클릭

2. **안내 따라가기**
   - 자동으로 진행됨
   - Firebase 로그인 창이 뜨면 Google 계정으로 로그인
   - "계속하시겠습니까? (Y/N)" → `Y` 입력

3. **완료!**
   - https://samsung-gfc.web.app 에서 확인

### 🍎 Mac

1. **터미널 열기**
   - `Cmd + Space` → "터미널" 입력

2. **스크립트 실행**
   ```bash
   cd ~
   ./6-one-click-deploy.sh
   ```

3. **안내 따라가기**
   - 계속하시겠습니까? (y/N): `y` 입력
   - Firebase 로그인 브라우저 자동 열림

4. **완료!**
   - https://samsung-gfc.web.app 에서 확인

## 🎁 추가 기능

### 자동으로 하는 것들

- ✅ Git, Node.js, npm 설치 확인
- ✅ Firebase CLI 최신 버전 자동 설치
