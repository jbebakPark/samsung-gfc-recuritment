## 🔴 핵심 문제: "막대 크기가 거꾸로 그렸다"

### 사용자 스크린샷 분석

**현재 보이는 것** (잘못됨):
- 왼쪽 (1~2년차, 회색) = **가장 큰 막대** ❌
- 가운데 (3~5년차, 파란색) = 중간 크기 ❌
- 오른쪽 (6년차+, 노란색) = **가장 작은 막대** ❌

**올바른 비율**:
- 왼쪽 (1~2년차) = **가장 작아야 함** (33%)
- 가운데 (3~5년차) = 중간 (69%)
- 오른쪽 (6년차+) = **가장 커야 함** (100%)

**결론**: 막대가 **완전히 거꾸로** 표시되고 있음!

## 🔍 근본 원인 분석

### 1. CSS 구조
```css
.chart-container {
    align-items: flex-end;  /* 막대를 아래에서부터 쌓음 ✅ */
}

.bar-fill {
    height: 33%;            /* 높이는 올바름 ✅ */
}
```

### 2. 문제의 핵심: 그라데이션 방향

```css
/* Before - 잘못된 그라데이션 */
.bar-fill {
    background: linear-gradient(180deg, #1428A0 0%, #034EA2 100%);
    /*                          ^^^^^^
                               위→아래 방향 ❌
    */
}
```

**시각적 효과**:
```
막대가 아래에서 위로 자라는데 (align-items: flex-end)
그라데이션은 위에서 아래로 그려짐 (180deg)
→ 시각적으로 거꾸로 보임!
```

### 3. 왜 거꾸로 보였나?

#### 실제 렌더링 (Before)
```
┌─────────────┐  ← 밝은 색 (#1428A0) 시작
│             │
│   1~2년차   │  height: 33%
│   4,800만   │
└─────────────┘  ← 어두운 색 (#034EA2) 끝
▔▔▔▔▔▔▔▔▔▔▔▔▔  바닥

결과: 위쪽이 밝아서 더 크게 보임 ❌
```

#### 이제 (After)
```
┌─────────────┐  ← 어두운 색 (#034EA2) 시작
│             │
│   1~2년차   │  height: 33%
│   4,800만   │
└─────────────┘  ← 밝은 색 (#1428A0) 끝
▔▔▔▔▔▔▔▔▔▔▔▔▔  바닥

결과: 아래쪽이 밝아서 작게 보임 ✅
```

## ✅ 해결 방법

### 그라데이션 방향 반전

#### Before (180deg = 위→아래)
```css
.bar-fill {
    background: linear-gradient(180deg, #1428A0 0%, #034EA2 100%);
}

.bar-fill.bar-1 {
    background: linear-gradient(180deg, #6c757d 0%, #495057 100%);
}

.bar-fill.bar-2 {
    background: linear-gradient(180deg, #1428A0 0%, #034EA2 100%);
}

.bar-fill.bar-3 {
    background: linear-gradient(180deg, #FFD700 0%, #FFA500 100%);
}
```

#### After (0deg = 아래→위)
```css
.bar-fill {
    background: linear-gradient(0deg, #1428A0 0%, #034EA2 100%);
}

.bar-fill.bar-1 {
    background: linear-gradient(0deg, #495057 0%, #6c757d 100%);
}

.bar-fill.bar-2 {
    background: linear-gradient(0deg, #034EA2 0%, #1428A0 100%);
}

.bar-fill.bar-3 {
    background: linear-gradient(0deg, #FFA500 0%, #FFD700 100%);
}
```

**핵심 변경**:
- `180deg` → `0deg` (방향 반전)
- 색상 순서도 반전 (어두운색 → 밝은색)

## 🎨 그라데이션 방향 이해

### 180deg (위→아래)
```
┌─────────┐
│ 밝은색  │ ← 0% (시작)
│    ↓    │
│    ↓    │
│ 어두운색 │ ← 100% (끝)
└─────────┘
```

### 0deg (아래→위)
```
┌─────────┐
│ 어두운색 │ ← 100% (끝)
│    ↑    │
│    ↑    │
│ 밝은색  │ ← 0% (시작)
└─────────┘
바닥
```

**Flexbox `align-items: flex-end`와 조합**:
- 막대가 **바닥에서 위로** 자람
- 그라데이션도 **바닥에서 위로** 가야 자연스러움
- → `0deg` 사용 ✅
