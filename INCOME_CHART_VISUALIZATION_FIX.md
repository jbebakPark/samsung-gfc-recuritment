## 🔴 문제점 (사용자 피드백)

### "납득이 안가는 그래프"

사용자가 제공한 스크린샷을 보면:
- **1~2년차 막대**: 매우 크고 높게 보임 (실제로는 33%여야 함)
- **3~5년차 막대**: 중간 크기
- **6년차+ 막대**: 상대적으로 작게 보임

**문제 원인**:
```css
/* Before - 문제가 있던 CSS */
.bar-fill {
    justify-content: flex-start;  /* 텍스트를 위쪽에 배치 */
    padding: 20px 10px;
}
```

이 설정으로 인해:
1. 텍스트가 막대 **위쪽**에 고정
2. 막대 높이가 33%여도 텍스트는 위쪽에 있어서 막대가 크게 보임
3. 시각적 혼란 발생

---

## ✅ 해결 방법

### 1. 텍스트 중앙 배치

#### Before
```css
.bar-fill {
    justify-content: flex-start;  /* 위쪽 정렬 */
    padding: 20px 10px;
}
```

#### After
```css
.bar-fill {
    justify-content: center;      /* 중앙 정렬 ✅ */
    padding: 15px 10px;
    min-height: 80px;             /* 최소 높이 보장 ✅ */
}
```

**효과**:
- 텍스트가 막대 **중앙**에 위치
- 막대 높이에 비례하여 텍스트도 상하로 이동
- 시각적으로 비율이 명확하게 보임

### 2. 텍스트 가시성 향상

#### Before
```css
.bar-label {
    font-size: 0.9rem;
    font-weight: 600;
}

.bar-value {
    font-size: 1.1rem;
    font-weight: 700;
}
```

#### After
```css
.bar-label {
    font-size: 1rem;              /* +0.1rem */
    font-weight: 700;             /* +100 */
    text-shadow: 0 2px 4px rgba(0,0,0,0.2);  /* 그림자 추가 ✅ */
}

.bar-value {
    font-size: 1.3rem;            /* +0.2rem */
    font-weight: 800;             /* +100 */
    text-shadow: 0 2px 4px rgba(0,0,0,0.2);  /* 그림자 추가 ✅ */
    letter-spacing: -0.5px;       /* 타이트한 간격 */
}
```

**효과**:
- 텍스트가 더 크고 굵게 표시
- 그림자로 배경에서 명확하게 구분
- 가독성 대폭 향상

## 🎨 개선 세부사항

### 1. justify-content 변경
- **Before**: `flex-start` (위쪽 정렬)
- **After**: `center` (중앙 정렬)
- **효과**: 막대 높이에 따라 텍스트 위치 자동 조정

### 2. min-height 추가
- **값**: 80px
- **이유**: 33% 높이의 막대도 텍스트가 충분히 들어갈 공간 확보
- **효과**: 작은 막대에서도 텍스트가 잘림 없이 표시

### 3. 텍스트 크기 증가
- **bar-label**: 0.9rem → 1rem (+11%)
- **bar-value**: 1.1rem → 1.3rem (+18%)
- **효과**: 멀리서도 명확하게 읽힘

### 4. text-shadow 추가
- **값**: `0 2px 4px rgba(0,0,0,0.2)`
- **효과**: 색상이 있는 배경에서도 텍스트가 명확하게 구분됨

### 5. padding 최적화
- **Before**: 20px 10px
- **After**: 15px 10px
- **효과**: 여백을 줄여 막대 크기를 더 정확하게 표현
