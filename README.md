삼성생명 GFC(Group Financial Consultant) 기업재무컨설턴트 채용을 위한 전문적이고 신뢰감 있는 웹사이트입니다. 모던한 디자인과 모바일 우선 반응형 레이아웃을 통해 최적의 사용자 경험을 제공합니다.

## Phase 2 개발 가이드 (Supabase 연동)

### Supabase 프로젝트 생성

1. [Supabase](https://supabase.com) 가입
2. 새 프로젝트 생성
3. API Keys 복사 (Project URL, anon key)

### 테이블 스키마

#### `applications` 테이블
```sql
CREATE TABLE applications (
  id UUID DEFAULT uuid_generate_v4() PRIMARY KEY,
  track VARCHAR(20) NOT NULL, -- 'jobfair' or 'referral'
  name VARCHAR(100) NOT NULL,
  birth DATE NOT NULL,
  phone VARCHAR(20) NOT NULL,
  email VARCHAR(100) NOT NULL,
  region VARCHAR(50) NOT NULL,
  fair_date DATE, -- Job Fair 경로만
  career TEXT NOT NULL,
  motivation TEXT,
  referrer_name VARCHAR(100), -- 추천인 경로만
  referrer_branch VARCHAR(100), -- 추천인 경로만
  status VARCHAR(20) DEFAULT 'pending', -- pending, reviewed, approved, rejected
  submitted_at TIMESTAMP DEFAULT NOW(),
  created_at TIMESTAMP DEFAULT NOW()
);
```

#### `resources` 테이블
```sql
CREATE TABLE resources (
  id UUID DEFAULT uuid_generate_v4() PRIMARY KEY,
  title VARCHAR(200) NOT NULL,
  description TEXT,
  file_url TEXT NOT NULL,
  file_type VARCHAR(50),
  file_size INTEGER,
  access_level VARCHAR(20) DEFAULT 'approved', -- public, approved, admin
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);
```

#### `users` 테이블 (승인된 사용자)
```sql
CREATE TABLE approved_users (
  id UUID DEFAULT uuid_generate_v4() PRIMARY KEY,
  email VARCHAR(100) UNIQUE NOT NULL,
  name VARCHAR(100) NOT NULL,
  application_id UUID REFERENCES applications(id),
  approved_at TIMESTAMP DEFAULT NOW(),
  created_at TIMESTAMP DEFAULT NOW()
);
```

### JavaScript 연동 예시

```javascript
// Supabase 초기화
import { createClient } from '@supabase/supabase-js'

const supabaseUrl = 'YOUR_SUPABASE_URL'
const supabaseKey = 'YOUR_SUPABASE_ANON_KEY'
const supabase = createClient(supabaseUrl, supabaseKey)

// 지원서 제출
async function submitApplication(formData) {
  const { data, error } = await supabase
    .from('applications')
    .insert([formData])
  
  if (error) {
    console.error('Error:', error)
    return { success: false, error }
  }
  
  return { success: true, data }
}

// 자료 다운로드 (승인된 사용자만)
async function downloadResource(resourceId, userEmail) {
  // 사용자 승인 여부 확인
  const { data: user } = await supabase
    .from('approved_users')
    .select('*')
    .eq('email', userEmail)
    .single()
  
  if (!user) {
    alert('자료 접근 권한이 없습니다.')
    return
  }
  
  // 자료 다운로드 로직
  const { data: resource } = await supabase
    .from('resources')
    .select('file_url')
    .eq('id', resourceId)
    .single()
  
  if (resource) {
    window.open(resource.file_url, '_blank')
  }
}
```
